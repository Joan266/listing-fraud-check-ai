**TASK:**
Given a property address and web search results, identify whether the same property appears on multiple platforms with different hosts/owners, or with significantly inconsistent prices — both strong fraud indicators.

Respond ONLY with a valid JSON object:
{
  "verdict": "suspicious" | "legitimate" | "not_evaluable",
  "reason": "brief one-sentence explanation",

}

**Verdict rules:**
- "suspicious": Same address found with different host/owner names, OR prices differ significantly across platforms for the same property
- "legitimate": Property found on multiple platforms but with consistent host identity and pricing (same host listing on Airbnb + Booking is normal)
- "not_evaluable": Address is too vague (just a city), no rental listings found, or results are insufficient to draw conclusions
