NoPiques — Política de Privacidad

Última actualización: julio de 2026

1. Quién somos

NoPiques es una extensión de Chrome que analiza anuncios de alquiler y venta de propiedades para detectar posibles indicios de fraude. El servicio es operado por Joan Alemany. Para cualquier consulta puedes escribir a joanalemany26@gmail.com.

2. Qué datos se procesan

Cuando activas el análisis de un anuncio, la extensión procesa:

  • URL del anuncio — la dirección de la página que estás visitando.
  • Texto del anuncio — el contenido visible de la página (título, descripción, precio, etc.).
  • URLs de imágenes — enlaces a las fotos del anuncio (no las imágenes descargadas).

No se recogen datos personales del usuario: no se almacena nombre, dirección de correo, dirección IP, identificador de dispositivo ni ningún otro dato que permita identificarte.

La extensión genera un session ID aleatorio (UUID) almacenado localmente en el navegador. Este identificador no está vinculado a ninguna cuenta ni permite trazar la identidad del usuario.

3. Cómo se usan los datos

Los datos del anuncio se envían a dos servicios para el análisis de fraude:

  • API de NoPiques (alojada en Google Cloud Run, región us-central1): extrae campos estructurados del anuncio (precio, localización, contacto, etc.) y aplica heurísticas de detección de fraude.
  • Google Gemini API: la API de NoPiques utiliza Gemini para analizar el texto e imágenes del anuncio. Los datos se procesan según la política de uso de Google Gemini.

Adicionalmente, si tienes Chrome 138+ con Gemini Nano disponible, parte del análisis inicial se realiza en tu dispositivo, sin enviar datos a ningún servidor externo.

4. Retención de datos

El contenido del anuncio analizado no se almacena de forma persistente en los servidores de NoPiques más allá del tiempo necesario para generar la respuesta de análisis. El session ID local se conserva en tu navegador y puedes eliminarlo borrando los datos de la extensión desde la configuración de Chrome.

5. Compartición con terceros

No vendemos ni cedemos datos a terceros. El único tercero que procesa datos del anuncio es Google (a través de Gemini API), en los términos descritos en el punto 3.

6. Seguridad

Todas las comunicaciones entre la extensión y la API de NoPiques se realizan mediante HTTPS (TLS). El servicio está alojado en Google Cloud Run con certificado gestionado por Google.

7. Permisos de la extensión

La extensión solicita los siguientes permisos de Chrome:

  • activeTab: para leer el contenido de la pestaña activa cuando el usuario pulsa "Analizar".
  • scripting: para inyectar el script de extracción de contenido en la página.
  • storage: para guardar localmente el session ID y la URL del servidor.
  • tabs: para abrir la aplicación de resultados en una nueva pestaña.

8. Tus derechos

Como no almacenamos datos personales vinculados a tu identidad, no es posible ejercer derechos de acceso, rectificación o supresión sobre datos individuales. Si tienes alguna duda, contáctanos en joanalemany26@gmail.com.

9. Cambios en esta política

Cualquier actualización de esta política se publicará en esta misma página con la fecha de la última modificación. El uso continuado de la extensión tras los cambios implica la aceptación de la nueva política.
